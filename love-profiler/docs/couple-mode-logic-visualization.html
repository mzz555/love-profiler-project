love-profiler · 双人落差测评引擎

两个人，一把落差仪
量出你们以为一致、其实差很远的地方

一对情侣分别独立作答、并互相预测对方的答案。引擎用纯代码算出两人在少数几个维度上的落差，并特别抓出盲区——你以为你懂，其实没猜中。它不评判合不合适，只给你们一份能一起聊的对话指南。

② 计算引擎 · 交互 playground

拖动滑块，看七步纯算流水线实时跑通

引擎是确定性纯代码：无 LLM、无 DB。下面每个维度有两位作答者的滑块（青=A，玫红=B）；标 apply_prediction 的维度还有"互猜"滑块（虚线幽灵标记）。所有读数都由 JS 忠实镜像 app/agents/couple_scoring/*.py 的公式实时算出。

九条反判决铁律（system prompt 硬编码）

1禁判决：不出现匹配度/合不合适/般配/注定/及格，落差只框定为"值得聊的话题"
2complementary 维度只写优势或中性，禁负面措辞
3topic_only / 未校准维度最多轻提，不严肃解读
4level_only 维度只描述各自节奏，不把差距说成问题
5每个落差先给中性事实（narrative_fact），再给开放式问题
6用 gap_level 语义（接近/有些不同/差得比较多），不念原始数字
7保护隐私，绝不暴露对方裸答案
8说中感来自忠实转述，不来自夸张煽情
9高摩擦组合中性化解读，不渲染成危险信号

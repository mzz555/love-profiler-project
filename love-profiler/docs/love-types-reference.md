## type_code 编码规则

`{依恋}-{边界}-{冲突}`：

- **依恋**（D1）：`S` 安全 / `MS` 中度安全 / `MA` 中度焦虑 / `A` 焦虑
- **边界**（D2）：`CL` 清晰 / `BL` 模糊
- **冲突**（D3）：`H` 健康 / `P` 问题

4 × 2 × 2 = 16 类。D4（爱的语言）和 D5（表达风格）作为子画像，不进入主类型编码。
